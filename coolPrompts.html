GENERATIVE PROMPTS:

This is a set of prompts for use with stable horde, or any other comparable AI txt2img generator.
negative prompt is listed after ###, weight must be consistent with this repo
These prompt should aim to be generative in nature and consistent in depictions or style
This means the prompt should:
- Aim to generate generic things, as opposed to generating images of specific already existing characters or buildings.
- Aim to consistently put out images with an artistically similar style.
- Aim to consistently put out images with a similar general focus. (stuff like a forest, or gemstones)
The goal of the prompts listed here is if people need something like a picture of a generic ruin, weapon, monster, etc for a game or roleplaying.
And also to provided sets of tags that can be added onto a description, to hone in a specific art-style.
